Learning Objectives

By the end of this session, learners will be able to:

  • Identify emotional and behavioural triggers using appraisal theory
  • Regulate reactions using CBT + mindfulness techniques
  • Apply the C.A.L.M. Model (MI + NVC based)
  • Use assertive communication during conflict

Why Conversations Become Difficult

Biological & Emotional

  • • Perceived threat (amygdala activation)
  • • High emotional load (Goleman)

Cognitive

  • • Cognitive distortions (CBT)
  • • Ambiguity + misunderstanding
"We respond to our interpretation — not the event itself."
- Lazarus & Folkman

The Reaction Loop

Trigger
Emotion
Thought
Reaction

Adapted from Beck’s Cognitive Model (CBT).
Key: You can't control triggers, but you can control reactions.

Technique #1

The 10-Second Pause

Based on Mindfulness-Based Stress Reduction (Kabat-Zinn)

Pause
Breathe
Label

Benefits: Reduces emotional hijack, increases clarity.

Technique #2

The C.A.L.M. Model

C Clarify the issue
A Acknowledge emotions
L Listen actively
M Move toward solutions

Inspired by Motivational Interviewing & Nonviolent Communication.

Scenario Practice

"Your colleague has delayed their part of a shared project twice."

Prompt: How would you start this conversation using C.A.L.M.?
